## app设置（手机）

### 设置音乐源

1. 找到左上角的三条杠，单击打开
2. 左侧栏中找到设置
3. 下滑找到音乐来源，点击“自定义源管理”
4. 点击导入，选择本地导入
5. 找到下载音乐源的文件夹，找到js文件逐次点击添加即可
6. 出现下面“导入成功”就可以了
7. 记得勾选其中的一个音乐源

### 参数设置可以参考电脑的参数设置
